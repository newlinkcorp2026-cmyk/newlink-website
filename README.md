# NEWLINK Website

공식 NEWLINK 기업 홈페이지 프로젝트

## Website

https://www.newlinkcorp.kr

## Overview

NEWLINK의 브랜드, 서비스, 구축 사례를 소개하고
고객 문의를 수집하기 위한 기업 웹사이트입니다.

## Features

### Corporate Website

* 회사 소개
* 서비스 소개
* 구축 사례
* 문의하기
* 반응형 웹 지원

### Contact Form

* 실시간 입력 검증
* 이메일 문의 접수
* Resend 연동
* Vercel Serverless API

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion

### Backend

* Vercel Serverless Functions

### Infrastructure

* GitHub
* Vercel
* Resend
